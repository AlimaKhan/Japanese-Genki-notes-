📒 Genki (LESSON 2)

Lesson 2:

Kore (これ)・Sore (それ)・Are (あれ)・Dore (どれ)

Examples

Example 1 (これ / This):
これはペンです。
Kore wa pen desu. → This is a pen.

Example 2 (これ / This):
これはりんごです。
Kore wa ringo desu. → This is an apple.

Example 3 (これ / This):
これはいぬです。
Kore wa inu desu. → This is a dog.

Example 4 (それ / That):
それはほんです。
Sore wa hon desu. → That is a book.

Example 5 (それ / That):
それはねこです。
Sore wa neko desu. → That is a cat.

Example 6 (それ / That):
それはおちゃです。
Sore wa ocha desu. → That is tea.

Example 7 (あれ / That over there):
あれはやまです。
Are wa yama desu. → That (over there) is a mountain.

Example 8 (あれ / That over there):
あれはくるまです。
Are wa kuruma desu. → That (over there) is a car.

Example 9 (あれ / That over there):
あれはがっこうです。
Are wa gakkou desu. → That (over there) is a school.

Example 10 (どれ / Which):
どれがあなたのかばんですか。
Dore ga anata no kaban desu ka? → Which one is your bag?

Example 11 (どれ / Which):
どれがサナさんのほんですか。
Dore ga Sana-san no hon desu ka? → Which one is Sana’s book?

Example 12 (どれ / Which):
どれがいちばんすきですか。
Dore ga ichiban suki desu ka? → Which one do you like the most?

✨ Practice Time

Try writing and speaking these sentences using これ・それ・あれ・どれ:

  1. Say: “This is my book.” (use これ and の)
  2. Say: “That is your bag.” (use それ and の)
  3. Say: “That (over there) is a school.” (use あれ)
  4. Ask: “Which one is your pen?” (use どれ and の)
  5. Ask: “Which one is Sana’s notebook?” (use どれ and の)
